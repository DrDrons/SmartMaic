1. ДОБАВЛЕНИЕ НОВОГО УСТРОЙСТВА:

    Для добавления нового устройства, нужно последовательно заполнить все поля для ввода.
    1) "Название Устройства" - по названию ясно, что в это поле вводится название устройства,
    и именно так программа будет называть добавленное устройство в дальнейшем.
    2) "IP-Адрес" - данное поле принимает IP-Айдрес устройства под которым оно находится в сети.
    3) "ID Устройства" - поле для ввода ID добавляемого устройства.
    4) "Значение одного импульса" - данное поле для ввода принимает значение одного импульса.
    То есть - сколько литров (в случае если счётчик считает воду) обозначается одним импульсом.
    Таких полей с одноимённым названием два (описываемое поле относится к ПЕРВОМУ импульсому входу),
    чтобы не перепутать их, воспользуйтесь всплывающими подсказками.
    5) "Единица измерения" - поле для обозначения единицы измерения того, что будет считать счетчик
    (данное поле относится к ПЕРВОМУ импульсному входу).
    6) "Значение одного импульса" - принимает аналогичные данные поля с одноимённым названием, НО
    относится к ВТОРОМУ импульсному входу.
    7) "Единица измерения" - поле для обозначения единицы измерения того, что будет считать счетчик
    (данное поле относится к ВТОРОМУ импульсному входу).
